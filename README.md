# ElhiKids Landing Page – Prueba con IA (Grok)

**Proyecto de prueba / experimento**  
Landing page para el servicio de arriendo de plazas blandas y juegos infantiles **ElhiKids** (Santiago, Chile), creada casi enteramente con ayuda de **Grok** (xAI) en febrero 2026.

### ¿Qué es esto?
- Es una landing page responsive, mobile-first, sin frameworks (solo HTML + CSS + JS vanilla)
- Diseñada pensando en alta conversión para cotizaciones por WhatsApp
- Tema infantil premium pero confiable (colores pastel, formas suaves, énfasis en seguridad y puntualidad)
- **Hecha principalmente conversando con una IA** (Grok) paso a paso durante varias iteraciones

### Tecnologías / características
- HTML5 semántico
- CSS puro (variables, mobile-first, responsive con media queries)
- JavaScript vanilla (smooth scroll, acordeón FAQ, formulario → WhatsApp, animaciones con IntersectionObserver)
- Google Fonts (Nunito)
- Lazy loading de imágenes
- Accesibilidad básica (aria-labels, foco visible, navegación por teclado)
- SEO básico + JSON-LD LocalBusiness
- Botón flotante WhatsApp con animación sutil
- Inspiración visual en landings chilenas del rubro (Rentaparty, NamuncuKids, Softplaycl, etc.)

### Estado del proyecto
✅ Prueba / Experimento  
❌ No es la versión oficial de ElhiKids  
❌ No tiene backend ni integración real con pagos/reservas  
❌ Imágenes son placeholders de Unsplash (reemplazar por fotos propias)
